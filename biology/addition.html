Дополнительная информация
Некоторые пакеты языка R
affy - пакет для исследования олигонуклеотидных последовательностей (проект Bioconductor) limma - пакет для исследования микро-array экспрессии генов (проект Bioconductor) clusterProfiler - пакет для функционального анализа генов WGCNA (weighted co-expression network analysis) - пакет для взвешенного сетевого анализа совместной экспрессии
Некоторые пакеты языка Python
NumPy — библиотека для работы с многомерными массивами и матрицами SciPy — библиотека, расширяющая возможности NumPy Scikit-learn — библиотека для машинного обучения Pandas — библиотека для обработки и анализа структурированных данных XGBoost - библиотека для градиетного бустинга Matplotlib - библиотека для визуализации данных с помощью двумерной и трехмерной графики Seaborn — библиотека для анализа данных и отображения сложных зависимостей с помощью графиков Pillow - библиотека для обработки изображений
