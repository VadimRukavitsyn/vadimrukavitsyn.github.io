Дополнительная информация
Некоторые пакеты языка R
affy - пакет для исследования олигонуклеотидных последовательностей (проект Bioconductor) limma - пакет для исследования микро-array экспрессии генов (проект Bioconductor) clusterProfiler - пакет для функционального анализа генов WGCNA (weighted co-expression network analysis) - пакет для взвешенного сетевого анализа совместной экспрессии glmnet - пакет для регуляризации LASSO-регрессии и эластической сети e1071 - пакет для статистического моделирования и машинного обучения kernlab - пакет для ядерного машинного обучения caret - пакет для обучения и использования классификационных и регрессионных моделей randomforest - пакет для классификации и обучения с использованием метода случайного леса nnet - пакет для создания нейронных сетей с одним скрытым слоем neuralnet - пакет для создания нейронных сетей со множеством скрытых слоев NeuralNetTools - пакет для визуализации и анализа нейронных сетей ggplot2 - пакет для визуализации данных ggpubr - пакет для создания графиков для публикации на основе ggplot2
Некоторые пакеты языка Python
NumPy — библиотека для работы с многомерными массивами и матрицами SciPy — библиотека, расширяющая возможности NumPy Scikit-learn — библиотека для машинного обучения Pandas — библиотека для обработки и анализа структурированных данных XGBoost - библиотека для градиетного бустинга Matplotlib - библиотека для визуализации данных с помощью двумерной и трехмерной графики Seaborn — библиотека для анализа данных и отображения сложных зависимостей с помощью графиков Pillow - библиотека для обработки изображений Hyperopt - библиотека для оптимизации гиперпараметров
