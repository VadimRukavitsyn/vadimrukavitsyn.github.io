Нейронные сети
Виды нейронных сетей
Convolution neural network (CNN) - сверточная нейронная сеть Artificial neuronal network (ANN) - искуственная нейронная сеть
Методы машинного обучения
K-nearest neighbors algorithm (KNN) - метод K-ближайших соседей Support vector machine (SVM) - метод опорных векторов Support vector machine-based recursive feature elimination (SVM-RFE) - основанное на методе опорных векторов удаление рекурсивных признаков Random forest (RF) - метод случайного леса Decision tree learning (DT) - метод дерева принятия решений Least absolute shrinkage and selection operator regression (LASSO-regression) - ЛАССО-регрессия Naive Bayes classifier (Bayes) - наивный Байесовский классификатор Attention mechanism - механизм внимания (различие относительной важности каждого компонента в последовательности) Gradient boosting (GB) - градиентный бустинг
Методы машинного зрения
Local binary patterns (LBP) - локальные бинарные шаблоны Histogram of oriented gradients (HOG) - гистограммы направленных градиентов
Методы интерпретации заключений модели
LIME (Local Interpretable Model-agnostic Explanations) - выбор начального набора данных и анализ колебаний SHAP (SHapley Additive exPlanations) — определение вклада каждого признака в конкретный прогноз
Оценка точности нейронных сетей
Robustness - надежность Accuracy - общая точность модели (по всем классам; (TP + TN) / (TP + TN + FP + FN)) Confusion matrix - матрица ошибок 1. True Positive (TP) — истинно-положительные объекты (относятся к определяемому классу, верно определены моделью) 2. True Negative (TN) — истинно-отрицательные объекты (не относятся к определяемому классу, верно определены моделью) 3. False Positive (FP) — ложно-положительные объекты (не относятся к определяемому классу, неверно определены моделью) 4. False Negative (FN) — ложно-отрицательные объекты (относятся к определяемому классу, неверно определены моделью) Precision - точность (отношение TP к TP + FP) Recall (sensitivity) - полнота (чувствительность; отношение TP к TP + FN) Specificity - специфичность (отношение TN к FP + TN) F1-score - оценка F1 (среднее гармоническое точности и полноты) Receiver operating characteristic (ROC) curve - кривая ошибок Area under the ROC curve (AUC) - площадь под кривой ошибок Cohen's kappa coefficient - статистический показатель для оценки согласованности оценок
Визуализация множеств
Диаграмма Эйлера - содержит не все конечные подмножества Диаграмма Венна - содержит все конечные подмножества
