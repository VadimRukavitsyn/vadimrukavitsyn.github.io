Нейронные сети

Accuracy - общая точность модели (по всем классам)

Confusion matrix - матрица ошибок

1. True Positive (TP) — истинно-положительные объекты (относятся к определяемому классу, верно определены моделью)

2. True Negative (TN) — истинно-отрицательные объекты (не относятся к определяемому классу, верно определены моделью)

3. False Positive (FP) — ложно-положительные объекты (не относятся к определяемому классу, неверно определены моделью)

4. False Negative (FN) — ложно-отрицательные объекты (относятся к определяемому классу, неверно определены моделью)

Precision - точность (отношение TP к TP + FP)

Recall - полнота (отношение TP к TP + FN)

Receiver operating characteristic (ROC) curve - кривая ошибок

Area under the ROC curve (AUC) - площадь под кривой ошибок
